TopView

The top of the panel displays a 2-dimensional representation of the currently loaded Map at the current editing level. Using the mousewheel when the panel is focused raises and lowers the editing level.

Left of the viewer panel are buttons for

    - level up [mousewheel up]
    - level down [mousewheel down]
    - cut a selected area [Ctrl+X] (respects Visibility toggles)
    - copy a selected area [Ctrl+C]
    - paste area at a selected tile [Ctrl+V] (respects Visibility toggles)
    - delete a selected area [Delete] (respects Visibility toggles)
    - fill a selected area with the currently selected tilepart from TileView [F]

Please note that pasting copied parts from one Map to a different Map usually requires that the allocated terrains of both Maps be identical.

Left-click to select a tile to edit, and a left-click drag motion will select multiple tiles over a rectangular area.

Double left-click selects the tilepart, in TileView, of the currently selected parttype.

Right-click places the currently selected tilepart from TileView into its proper tilepart type-slot along the bottom of the viewer, respecting both the clicked tile position on the grid and the currently selected parttype shown beneath the grid.

Double right-click clears the tilepart of the currently selected parttype.

The bottom of the panel displays the four parttype slots into which tileparts may be placed. The currently selected parttype has a light blue background.

Left-click selects that parttype's slot as ready to have a tilepart assigned to it.
    [1] - Floor
    [2] - Westwall
    [3] - Northwall
    [4] - Content

Double left-click will select the tilepart that's currently in the slot in TileView [T].

Right-click places the currently selected tilepart from TileView into the slot of selected tile(s) [Enter].

Double right-click clears the parttype slots of selected tile(s) [Shift+Delete].

To the right of the four parttype slots is a fifth slot that shows TileView's currently selected tilepart. Click on the sprite or press [Q] to auto-select its proper parttype slot.

The toolstrip buttons have the same functionality as those in MainView.

The color-blobs that appear on the Map are simplistic representations of the various parts that have been placed on tiles.

Note that [Ctrl+S] can be used to save the Map when the center panel has focus.

[F5] toggles TileView.
[F6] closes TopView or toggles TopView if TopRouteView is focused.
[F7] toggles RouteView.
[F8] closes TopRouteView or toggles TopRouteView if TopView is focused.

[Ctrl+F5] through [Ctrl+F8] focuses the respective viewer.
[Shift+F5] through [Shift+F8] focuses MainView.

[F11] minimizes all viewers.
[F12] restores all viewers.

[Esc] focuses the panel. That is, it defocuses any other control. It can also be used to abort a multi-tile selection.

[Ctrl+Q] quits MapView II


Keyboard navigation

The numeric keypad - that is, the arrow keys and [Home], [End], [PageUp], [PageDown] - select the tile in the chosen direction. The add and subtract keys on the numeric keypad lower and raise the current level respectively.

[Shift] can be held down to create a selection area.


Visible menu

These four operations toggle the visibility of the four possible tilepart types for TopView and MainView. They can be used for checking out flooring that's concealed behind a wall, for example. When a type is toggled off, the background color of its slot changes to dark gray - the parts are still modifiable in the parttype slots below the panel, however Cut/Paste/Delete operations consider visible parttypes only. A Copy operation copies all selected tileparts regardless of visibility.

The visibility settings have no effect when saving a Map file, all parts are saved whether they're currently visible or not. Note that the visibility of tileparts can also be toggled on and off with the function keys F1..F4.


Options [Ctrl+O] - opens and closes the Options for TopView. Descriptions for the entries appear at the bottom of that window.
