Requirements

  Windows 32- or 64- bit OS
  .NET 4.5.1
  ~1 mb diskspace plus a few megs of RAM

Note that .NET will inflate RAM usage exponentially.

An installation of either XCOM UFO or TFTD. The only files that are absolutely required from the stock game are the CURSOR.PCK/TAB files. Configuring MapView to work on your system will be disallowed if those files are not found in the UFOGRAPH subfolder. Other than that, it's certainly handy to have all the stock Maps, Routes, and Terrains - a default tileset configuration is supplied to display them with MapView.

Maps can be created from scratch, however, with only the cursor-sprite available. But you'd still need to create your own PCKs, TABs, and perhaps MCDs if you don't have the stock resources.
